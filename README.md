# eleanarinaudo.github.io

My personal portfolio website — **[eleanarinaudo.github.io](https://eleanarinaudo.github.io/)**

A single-page portfolio built with **Angular** and **TailwindCSS**: who I am, what I
work on (AI/ML, LLMs, data engineering) and how to get in touch.

## Tech stack

- **Angular** · TypeScript
- **TailwindCSS**
- Hosted on **GitHub Pages**

## Run locally

```bash
npm install
ng serve
```

Then open <http://localhost:4200>.
